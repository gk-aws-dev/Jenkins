- we can mention label in our pipeline. with help of the label we are mentioning the agent on which the pipeline will run.

```
pipeline {
	agent {
			label 'linux'
	}
	stages {
		stage ("Stage-1"){
			steps {
			    echo "Hello World"
			}
		}
		stage ("Stage-2"){
			steps {
			    echo "Hello World two"
			}
		}
		stage ("Stage-3"){
			steps {
			    echo "Hello World three"
			}
		}
	}
}
```
