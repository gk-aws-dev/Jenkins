
- Here is the example of the multistage pipeline. we can define the separate stage for the each work.

```
pipeline {
	agent any
	stages {
		stage ("Stage-1"){
			steps {
			    echo "Hello World"
			}
		}
		stage ("Stage-2"){
			steps {
			    echo "Hello World two"
			}
		}
		stage ("Stage-3"){
			steps {
			    echo "Hello World three"
			}
		}
	}
}
```
